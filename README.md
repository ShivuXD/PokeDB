# PokéDB

A simple Pokédex web application built with React and the PokéAPI.

This project started as a way for me to practice React after completing my first API-based project. Instead of following a tutorial, I built it step by step to better understand component design, state management, API integration, and responsive UI development.

## Features

* Browse Pokémon using data from the PokéAPI
* Search Pokémon by name
* Displays Pokémon image, type, height, and weight
* View base stats including:

  * HP
  * Attack
  * Defense
  * Speed
* Responsive card-based layout
* Pokémon-themed background

## Tech Stack

* React 19
* JavaScript (ES6+)
* CSS3
* PokéAPI

## Installation

Clone the repository:

```bash
git clone https://github.com/ShivuXD/PokeDB.git
```

Navigate to the project folder:

```bash
cd PokeDB
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

Open your browser and visit:

```txt
http://localhost:3000
```
